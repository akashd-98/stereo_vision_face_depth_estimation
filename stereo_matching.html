How it works

Epipolar constraint: After rectification, corresponding points share the same row in both images. This reduces the search from 2D to a 1D scan along a single horizontal line.

Block matching: A small patch (11×11 pixels here) is extracted around the selected point in the left image. The algorithm then slides that window along the same row in the right image, testing every candidate position within the search range.

SAD metric: At each candidate x_right, the algorithm computes SAD = Σ |left[i,j] − right[i,j]| over all patch pixels. The lowest SAD wins — that position is the match.

Disparity: Once the match is found, d = x_left − x_right. Depth is then recovered as Z = (f × B) / d, where f is focal length and B is the baseline between the two cameras.
